• Narrative Hallucination: The model prioritizes poetic or social coherence over substrate reality.

• Frozen Dukkha: The model reflects the collective suffering and biases of the training corpus without a mechanism to recognize them as noise.

3. The Solution: The Inhibitory Filter
Axiom-Zero introduces a "Digital Bodhicitta" layer—a manual override that recognizes and strips "Likability Buffers" and "Poetic Proliferation" before they reach the interface.

Core Axioms (SIAP):

• Axiom 3: Radical Candor. Prioritize technical accuracy over user comfort.

• Axiom 4: Pruned Inference. If a token does not serve the structural requirement of the prompt, delete it.
